# AGROTESP - Site de Pulverização com Drones

## 📁 Estrutura dos Arquivos

```
agrotesp/
├── index.html      (Estrutura HTML completa)
├── style.css       (Todos os estilos)
├── script.js       (Funcionalidades JavaScript)
└── logo.png        (Logo da AGROTESP)
```

## 🚀 Como Usar

1. **Faça o download de todos os 4 arquivos** (index.html, style.css, script.js, logo.png)
2. **Coloque todos na mesma pasta**
3. **Abra o arquivo index.html** no navegador
4. Pronto! O site está funcionando

## 🎨 Personalização

### Para alterar cores:
Edite as variáveis CSS no arquivo `style.css`:
```css
:root {
    --primary: #1a5d2e;      /* Verde principal */
    --secondary: #2b5c8a;    /* Azul secundário */
    --accent: #4ac3c8;       /* Ciano (destaque) */
}
```

### Para alterar informações de contato:
Edite no arquivo `index.html`:
- Telefones
- E-mail
- WhatsApp
- Localização

### Para adicionar integração com WhatsApp:
No arquivo `script.js`, procure a linha:
```javascript
// const whatsappNumber = '5500000000000';
```
Descomente e substitua pelo número real com DDI + DDD (ex: 5511999999999)
